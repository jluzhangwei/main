LLDP 拓扑工具使用手册

一句话：导入/追加 LLDP CSV，自动去重生成拓扑；支持分层/分散布局、过滤/搜索、多选/右键操作、邻接高亮；可导入导出 PNG/JSON/坐标，支持多链路合并切换。

启动方式

  • 仅离线导入 CSV 直接用浏览器打开 lldp.html 即可使用原有「导入 CSV / 追加 CSV / 粘贴 CSV」功能。
  • 启用 SQL 生成 CSV 需要启动本地服务，再通过 http://127.0.0.1:18080/lldp.html 访问页面。
  • 依赖安装 在虚拟环境执行：pip install fastapi uvicorn pymysql。
  • 数据库配置 在项目目录复制 .env.mysql.example 为 .env.mysql 并填写只读账号密码。
  • 启动命令 在 /Users/zhangwei/python/lldp_topology 运行：DB_ENV_FILE=/Users/zhangwei/python/lldp_topology/.env.mysql python lldp_sql_service.py。

一键启动（推荐）

  • 脚本路径 /Users/zhangwei/python/lldp_topology/start_lldp_service.sh
  • 启动服务 cd /Users/zhangwei/python/lldp_topology && ./start_lldp_service.sh start
  • 查看状态 ./start_lldp_service.sh status（包含环境检查、进程 PID、18080 端口监听信息）
  • 查看日志 ./start_lldp_service.sh logs
  • 重启服务 ./start_lldp_service.sh restart
  • 停止服务 ./start_lldp_service.sh stop
  • 访问页面 启动成功后打开 http://127.0.0.1:18080/lldp.html

SQL 生成 CSV（start_hostname）

  • 入口 页面「导入数据」区域新增：输入设备名 + 按钮「SQL 生成并建图」。
  • 输入 只需填写 start_hostname，例如 ID-DCI2-GENERAL.CONNECTION-R003-BRIDGE-02。
  • 行为 点击后会调用后端 SQL，生成临时 CSV，并自动载入页面生成拓扑。
  • 下载 可点击「下载临时 CSV」获取本次查询结果文件。
  • 兼容 不影响原有手动导入 CSV 功能，两种方式可并存使用。
  • 临时文件位置 服务写入 tmp_csv/ 目录。

CLI 递归导入（SMC 跳板）

  • 入口 导入方式选择 CLI 导入，点击「导入输入」。
  • 输入项 设备地址 + 递归次数（1-5，默认 3）。
  • SMC 参数 弹窗可填写设备账号、跳板地址/端口、SMC 命令、超时参数；优先使用弹窗值，其次回退到 .env.mysql。
  • 本地记忆 弹窗会在浏览器本地保存上次输入（密码除外），下次自动回填。
  • 登录方式 复用 Netlog 项目的 SMC 两阶段登录思路（先跳板 shell，再 SSH 目标设备）。
  • 执行命令 自动尝试：show lldp neighbor details / dis lldp neighbor（含常见变体）。
  • 递归规则 起始设备记为 depth=1，邻居设备继续 depth+1；已查询过的源设备不重复查询，直到达到递归次数。
  • 输出 提取 LLDP 邻接关系并生成临时 CSV，再自动载入页面生成拓扑。
  • 调试日志 每次 CLI 查询会生成调试日志文件，可在弹窗点击「下载 CLI 调试日志」获取。
  • 必要配置 在 .env.mysql 中补充：CLI_DEVICE_USERNAME、CLI_DEVICE_PASSWORD、SMC_JUMP_HOST、SMC_COMMAND。

数据导入

  • 导入 选择或拖拽 CSV，或粘贴文本；自动去重（按 localhostname/localinterface/remotehostname/remoteinterface）。
  • 列映射 默认自动匹配，列名异常时展开「列映射」下拉手动调整。
  • 追加 使用「追加 CSV」添加新设备，保留已有节点坐标不动。
  • 去后缀 选项可去掉 hostname 域名后缀。

布局与分散

  • 布局模式 仅强分散（默认 fCoSE）、上下分层、圆形。
  • 分散强度 0.1–1 滑杆，默认 0.5。
  • 分层关键字 逗号分层，斜杠 / 表示同层同义词，& 表示同层内全部命中才算匹配。示例：EDGE&CORE,CORE,AGGR/SPINE,LEAF/tor。
  • 上下分层 选择布局模式为「上下分层」或点击「应用分层」后「重新布局」。

显示与交互

  • 标签 节点标签缩略/全名切换；端口标签可显示/隐藏。
  • 过滤 逗号分隔关键词，命中节点隐藏并同步隐藏相关边。
  • 搜索 逗号分隔关键词，回车批量选中匹配节点。
  • 多选 Ctrl/Cmd 或 Shift；「仅显示邻接」高亮选中及其邻接并暗化其他。
  • 右键菜单 固定/取消固定、删除（可恢复）、复制名称。
  • 删除/恢复 「删除选中」移除节点及边，「恢复删除设备」撤销最近删除。

链路显示

  • 多链路合并 按钮「合并多条链路」可在合并/拆分模式间切换：同一对节点的多条链路合并为单条并显示计数。

坐标与导出

  • 坐标 「导出坐标」保存当前节点位置 JSON；「导入坐标」直接应用到拓扑（再居中缩放）。
  • 导出 PNG（修复空白问题，使用 blob）、JSON（当前画布状态）。

常用操作流

  • SQL 模式：启动服务 → 打开 http://127.0.0.1:18080/lldp.html → 输入 start_hostname → SQL 生成并建图。
  • 导入或追加 CSV → 选布局模式/分层关键字 → 生成或重新布局。
  • 调整分散强度、标签显示；用过滤/搜索快速筛选；多选 + 右键固定或删除。
  • 需要复用位置时导出坐标，下次导入坐标；若链路过多可开启「合并多条链路」。
